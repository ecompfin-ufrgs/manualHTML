    Depois de estruturar a página, para saber a tag adequada para o conteúdo que deseja criar, veja a seguinte página Tags.

    Partes básicas de um documento HTML

    1. header - tag header
    2. navigation bar - tag nav
    3. main - tag main
    4. sidebar - tag aside
    5. footer - tag footer

    O conteúdo do documento HTML exposto em <main> pode ser subdividido conforme o tipo de documento (artigo, página web etc) com as seguintes tags:

    • article - semântica
    • section - semântica
    • div

    Assim, o main pode ter vários articles ou um único article se o documento for ele mesmo um artigo. Alternativamente, um main pode ter várias section e, mesmo uma section, pode ter vários article ou um article várias section a depender de como for feita a divisão do documento.

    Minha recomendação é só usar section dentro de article ou dentro do main, mas não colocar article dentro de section.

    Considere também o uso da tag <span> para marcar parte do conteúdo de uma tag para posterior formatação. Esta tag, porém, não é semântica e, na medida do possível, deve ser evitada, preferindo-se tags semânticas para o mesmo conteúdo. Igualmente, a tag <div> a qual serve como divisória não semântica de texto em oposição a article,section e outras tags estruturantes como nav.

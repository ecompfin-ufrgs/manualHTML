    Metadata

    Metadata são dados que descrevem outros dados. No caso, os metadados vão descrever os dados contidos na página, isto é, o conteúdo dos elementos

    .

    De modo geral, o padrão da tag meta é o seguinte: <meta name="conteúdo" content="conteúdo" >, onde existem vários valores para os atributos name e content, mas os mais comuns são author e description.

    Incluindo citações dentro do parágrafo

    Citações dentro do parágrafo

    Use o elemento <q>. Este elemento admite o atributo <cite>, mas ele não serve para nada a menos de permitir apontar código JS para o elemento.

    Citações em bloco

    Use o elemento <blockquote>. Este elemento admite o atributo <cite>, mas ele não serve para nada a menos de permitir apontar código JS para o elemento.

    Observação

    O elemento <cite> serve apenas para mostrar que o elemento é um título de uma página. Se desejar apontar um link para o elemento, é necessário usar a tag <a>.

    Existem várias opções e a referência é MDN Web Docs. Listo abaixo, em ordem de preferência, outras opções mencionadas ou não pela Mozilla.

    Observação

    O HTML é uma linguagem que possui a capacidade de descrever estruturas de dados chamadas elementos com sintaxe formada pelas tags. Estes elementos possuem um conteúdo que o valor do dado armazenado. O JS - que é uma linguagem de programaçaõ que suporta preferencialmente do paradigma funcional - fornece as funções que unidas aos elementos HTML formarão objetos no sentido de programação orientada a objetos, ou seja, as funções JS são os métodos dos objetos da linguagem orientada a objetos HTML+JS. Ou seja, tudo se passa como se os atributos dos objetos fossem escritos com HTML e seus métodos com JS de tal forma que ambas as linguagens se complementam para formar uma única linguagem de programação orientada a objetos.
